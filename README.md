# Intervew Preparation

A structured collection of study notes, revision guides, examples, and cheat sheets across multiple technical subjects.

This repository is designed as a personal learning hub for interview preparation, coding practice, placement revision, and long-term reference. Each subject is organized into its own folder with topic-wise markdown files and GitHub-friendly navigation.

## Subjects

| Subject | Description | Status |
|---------|-------------|--------|
| [SQL Notes](SQL-Notes/README.md) | SQL fundamentals, operators, functions, joins, subqueries, window functions, database objects, DML, and cheat sheets. | Available |
| More subjects coming soon | Add new folders for additional subjects such as DBMS, OS, CN, DSA, Python, Java, System Design, and more. | Planned |

## How To Use

1. Open the subject folder you want to study.
2. Start with that subject's `README.md`.
3. Follow the recommended study path.
4. Use cheat sheets for fast revision.
5. Practice examples and patterns after each topic.

When adding new notes:
