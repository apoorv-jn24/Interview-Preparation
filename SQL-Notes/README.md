## Recommended Study Path

1. Start with SQL Introduction, SELECT, WHERE, ORDER BY, and DISTINCT.
2. Learn operators next: LIKE, BETWEEN, IN, IS NULL, and Boolean logic.
3. Move to aggregate functions, GROUP BY, and HAVING.
4. Study joins, subqueries, CTEs, and window functions.
5. Review database objects and DML safety.
6. Finish with the LeetCode and interview cheat sheets.
